# Vue Contact Demo

Vue 3 implementation of the Contact Finder demo using A2UI.

## Prerequisites

- Node.js 18+
- Python 3.10+ with UV package manager (for the agent)

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the demo (starts both agent and web server):
```bash
npm run demo
```

Or run them separately:

```bash
# Terminal 1: Start the agent
npm run serve:agent

# Terminal 2: Start the web server
npm run dev
```

3. Open http://localhost:4000 in your browser

## Features

- Search for contacts by name
- View contact details rendered through A2UI
- Responsive design with A2UI theming
